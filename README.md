# FreeRadius_Frontend

Легкий и быстрый веб-интерфейс на PHP для управления пользователями и устройствами (NAS) во FreeRADIUS с использованием SQLite.

## Возможности
- 🔐 Авторизация администратора (через БД).
- 👥 Управление пользователями (добавление, удаление, просмотр).
- 🛰 Управление оборудованием (NAS/Clients) через базу данных.
- 📂 Массовый импорт и экспорт (CSV) пользователей и устройств.
- 📋 Журнал действий администратора и лог аутентификаций FreeRADIUS.
- 🔄 Автоматическая ротация логов раз в сутки.
- 🐳 Полная поддержка Docker.

## Быстрый старт

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com
   cd FreeRadius_Frontend
   ```

2. **Подготовьте права и базу**:
   ```bash
   mkdir -p data config/radius www
   chmod -R 777 data www
   # Инициализируйте пустую базу (если нет своей)
   touch data/radius.db && chmod 666 data/radius.db
   ```

3. **Запустите проект**:
   ```bash
   docker-compose up -d
   ```

4. **Первый вход**:
   Откройте `http://localhost:8333`.
   - Логин: `admin`
   - Пароль: `admin`
   *(Смените пароль сразу после входа в верхней части страницы!)*

## Настройка FreeRADIUS
Чтобы FreeRADIUS читал данные из SQLite, убедитесь, что в `./config/radius/mods-available/sql` указан драйвер `rlm_sql_sqlite` и путь к базе `/var/lib/sqlite/radius.db`.
